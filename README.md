# spring-boot-learning-hub

A collection of hands-on Spring Boot demo projects designed to help developers learn, experiment, and master essential backend concepts.
This repository serves as a practical reference for integrating popular tools, applying clean architecture patterns, and exploring real-world features in Spring Boot applications.

# Showcases

- [Liquibase Demo](./liquibase-demo)
  - Liquibase Integration
  - Code Generator Demo

- [Soap Web Services](./soap-web-services)
    - Produce and Consume SOAP Web Services using jaxws
    - Produce and Consume SOAP Web Services using Spring-WS
    - Apply WS-Security using Spring-WS

- [Education Hub](./EducationHub)
  - Spring Security: For securing the application. 
  - Spring Cache (Caffeine): For caching JWT tokens. 
  - OpenAPI/Swagger: For API documentation.

- [Spring Microservices](./spring-microservices)
  - Service Discovery with Netflix Eureka
  - API Gateway with Spring Cloud Gateway
  - Circuit Breaker with Resilience4j
  - Spring Cloud Config for centralized configuration management

- [OAuth2 Demo](./OAuth2DemoApp)
  - OAuth2 Authorization Server
  - JWT Token Management

- [Spring Boot Testing](./learn-spring-boot-testing)
  - Unit Testing with JUnit and Mockito
  - Integration Testing with Spring Boot Test
